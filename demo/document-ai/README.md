# 多模态文档智能 Demo 包

## 方向选择

这套演示内容统一聚焦在一个方向：

**多模态文档智能（Document AI）**

这个方向很适合 Gist，因为它能自然覆盖以下功能：

- 链接导入
- OCR 截图识别
- 论文与资料管理
- 灵感记录
- 项目背景管理
- AI 问答
- 双文献对比
- 项目级 AI 总结
- Markdown 导出
- 比赛追踪

## 这套 Demo 包包含什么

- [project_background.md](./project_background.md)
  - 可直接作为“项目研究背景”上传到项目里的 Markdown
- [seed_resources.md](./seed_resources.md)
  - 已整理好的论文、资料、基准、比赛入口
- [demo_flow.md](./demo_flow.md)
  - 一套按功能走通的演示脚本

## 建议你在应用里创建的项目名

推荐项目名：

`多模态文档智能与科研知识管理`

## 这个方向适合演示什么

### 1. 资料采集

- 导入 arXiv 论文链接
- 导入 OpenReview 论文页
- 导入官方比赛页面
- 导入基准与数据集说明页

### 2. OCR 与图像理解

- 截图论文页或比赛页做 OCR
- 用图片导入表格、图示或排行榜信息

### 3. 文档管理

- 把论文、资料、比赛、灵感统一挂到同一个项目
- 用标签、摘要、项目归属进行整理

### 4. AI 能力

- 对单篇论文做问答
- 对两篇论文做对比
- 基于项目背景做项目总结
- 基于灵感反查相关论文或资料

### 5. 导出与汇报

- 导出项目总结
- 导出双文献对比结果

## 推荐的项目叙事

你可以把整套系统包装成这样一句话：

**这是一个围绕“多模态文档智能”研究主题构建的移动端研究助手，用来统一管理论文、资料、比赛、灵感和项目背景，并通过 AI 完成解析、问答、比较与总结。**
