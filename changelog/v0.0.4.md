# v0.0.4 - 2026-03-15

## 变更主题
完成主界面底部快速采集入口修复，补齐 `链接` 与 `扫描` 的原生桥接能力，并优化无悬浮窗权限时的用户提示流程。

## 本次完成内容
- 修复主界面底部 `+` 面板中 `链接` 按钮无响应的问题。
- 修复主界面底部 `+` 面板中 `扫描` 按钮无响应的问题。
- 为 WebView 注入的 `AndroidInterface` 新增快速采集桥接入口：
  - `openLinkCapture()`
  - `startScanCapture()`
- 新增 `QuickCaptureBridge`，统一处理快速采集相关动作的主线程派发与权限判断。
- 复用现有悬浮窗服务能力，使主界面 `链接` 可直接进入原生链接采集流程。
- 复用现有截图服务能力，使主界面 `扫描` 可直接进入原生截图识别流程。
- 优化 `链接` 入口的权限交互：
  - 已有悬浮窗权限时直接进入链接采集
  - 无悬浮窗权限时先弹出解释性提示，再由用户决定是否前往系统授权页
- 为快速采集桥接补充回归测试，覆盖：
  - 链接桥接
  - 扫描桥接
  - 语音桥接
  - HTML 侧按钮接线
